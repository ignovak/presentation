Simple presentation engine

Click to continue -->

You can use Left or Right arrow keys

And others...

PgUp, PgDown, Home, End, Space

This engine supports

  • Different tags
    • Lists
    • Code
    • Blockqoute
    • Images
    • Tables
    • Links
  • Navigation (with mouse, keyboard and location hash )
  • Automatic font-size tuning

It doesn't supports (yet)

  1. Javascriptlessness. But this page does!
  2. Code highlighting
  3. Markdown
  4. Transform animation effects
  5. ...And other things impress.js does

Usage

Put each page of your presentation into an <article> tag.

Wrap pages with some element.

<section id='presentation'>

  <article>
    <h1>Simple presentation engine</h1>
    <p>Click to continue</p>
  </article>

</section>

Usage

Then include jquery.js, presentation.js and this code:

$(function() {
  $('#presentation').present({
    prevBtn: '.btn.prev',
    nextBtn: '.btn.next',
    fadeDelay: 300 // default: 0
  });
});
