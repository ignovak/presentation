Simple presentation engine

Click to continue -->

You can use Left or Right

And either PgUp or PgDown

This engine supports

  • Lists
  • Code
  • Blockqoute
  • Images

It doesn't supports (yet)

  1. Code highlighting
  2. Markdown
  3. Transform effects
  4. ...And other things impress.js does

Usage

<section id='presentation'>
          
              <article>
                <h1>Simple presentation engine</h1>
                <p>Click to continue</p>
              </article>
          
            </section>

Usage

Then include jquery.js, presentation.js and this code:

$(function() {
            $('#presentation').present({
              prevBtn: '.btn.prev',
              nextBtn: '.btn.next',
              fadeDelay: 300 // default: 0
            });
          });
