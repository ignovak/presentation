Presentation.jscss Github

Simple presentation engine

Use Tab and Shift+Tab for navigating

This engine supports

  • Different tags
    • Lists
    • Code
    • Blockqoute
    • Images
    • Tables
    • Links
  • Navigation with keyboard
  • Javascriptlessness

It doesn't supports (yet)

  1. Code highlighting
  2. Markdown
  3. Transform animation effects
  4. ...And other things impress.js does

Usage

Put each page of your presentation into an <article> tag.

Wrap pages with some element.

<section id='presentation'>

  <article>
    <h1>Simple presentation engine</h1>
    <p>Click to continue</p>
  </article>

</section>
